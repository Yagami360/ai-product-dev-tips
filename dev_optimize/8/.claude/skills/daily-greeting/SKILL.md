# daily-greeting

短い挨拶メッセージを出力するだけの、最小構成のサンプルスキル。
開発作業に依存せず必ず動くため、Routine（定期実行）の動作確認用に使える。

## 手順

1. 「おはようございます」で始まる短い挨拶を1つ生成する。

2. ひとことの前向きな応援メッセージを添えて出力する。
