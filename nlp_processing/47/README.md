# Google ADK を使用して簡単なマルチ AI Agent を作成する（プロンプト指定のみの AI Agent）

## 方法

1. Google ADK の Python パッケージをインストールする

    ```bash
    pip install google-adk
    ```

1. Google ADK コマンドでプロジェクトのテンプレートを作成する

    ```bash
    adk create multi_agent
    ```
